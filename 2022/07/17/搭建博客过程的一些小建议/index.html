发表于2022-07-17|更新于2022-07-17
|阅读量:

#1为什么要搭建个人博客？

目前研0，由于是已经毕业了才考研的，目前在家等待开学，期间提前学习下要用的软件，在b站大学学习c语言的时候，老师推荐写blog，记录笔记，也可以提高写作能力。学的过程中，有的是发csdn，也有的大佬自己搭建网站。可惜我是在学完后，才想着要写blog，于是就有了这篇文章。

总共花了2天左右吧，期间被各种问题弄崩溃过：1用的系统不一样，有些代码不能直接用。2有些文章过于旧，软件版本不一样。3有些操作过于快，对于小白不太友好。

2软件安装

2.1 安装node.js

官网下载：Node.js 下载与自己系统配置的版本

一直默认安装就可以了

2.2 检查npm -v node -v 安装完成没有，成功出现以下界面

2.3 安装git

官网下载：Git - Downloading Package 比较慢，不建议

淘宝镜像：https://npm.taobao.org/mirrors/git-for-windows/ 比较建议

3搭建博客

3.1 创建blog本地文件和安装hexo

在某个盘创建blog本地文件

cmd进入到这个文件夹的目录（管理员身份打开）

输入npm install -g hexo-cli 进行安装hexo

输入hexo -v，验证是否成功

3.2 hexo 配置

接着上面继续操作，cmd窗口输入 hexo init（初始化hexo）

完成后在myblog文件内生成如下内容

按照顺序如下输入：

hexo clean # 清空已有hexo网站文件

hexo g(or generate) # 依据网页文本与新的CSS样式生成新网站文件

hexo s(or server) # 启动本地服务器，可以在localhost:4000查看网站修改效果

这时候访问http://localhost:4000/出现如下，则证明本地网站大家成功。

4主题更改

hexo主题网站： Themes | Hexo挑自己喜欢的点击进去，点下图中红色圈圈里的按钮下载，解压后放在MyBlog文件中的themes中，并命名为所下载的主题名（例如prince）

接着打开MyBlog文件中的_config.yml，找到如下圈圈中代码，修改为对应的主题名（我自己使用的是butterfly这个主题）

最后重新在cmd中依次输入 hexo clean

                                               hexo g

                                               hexo s

刷新网页，即可用上新主题
